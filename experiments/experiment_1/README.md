# Experiment 1 — CNN classification

This experiment evaluates the four-class CNN on the finalized 50-image reviewed set.

- scripts: `scripts/experiment_1_classification.py`
- optional excluded robustness protocol: `scripts/experiment_1b_color_robustness.py`
- final reviewed predictions: `raw_results/cnn_predictions_primary_reviewed.csv`
- publication package: `../../results/Experiment_1_Final_Publication/`

The optional color-robustness study was not finalized and is not part of the reported main results.
Raw AFM images are not redistributed. Supply `--test-dir` and, when appropriate, `--manifest` to rerun the script on an authorized local copy.
